Detailed Description

Originally created by Patrick Anderson. Modified by Samuel Senior on 10/03/2017. "DHT" evaluates the forward and backward discrete Hankel transform. Based on Fisk, Computer Physics Communications, 43 (1987). Complex datatype used here, should really template/overload.
